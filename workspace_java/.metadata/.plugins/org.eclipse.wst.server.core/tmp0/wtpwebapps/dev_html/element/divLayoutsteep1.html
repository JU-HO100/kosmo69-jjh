시맨틱웹

기계적인 검색 엔진은 어떤 태그가 어떤 기능을 하는지 분별할 수 없고 웹 페이지의 데이터를 효율적으로 추출할 수 없다. 이를 해결하고자 특정한 태그에 의미를 부여해서 웹페이지를 만드는 시도가 시작 되었다.

시맨틱 구조

geader - 헤더를 의미하는 태그 nav - 내비게이션을 의미하는 태그 aside - 사이드의 공간을 할당할수 있는 태그 section - 여러 중심 내용을 감싸는 공간을 의미 article - 글자가 많이 들어가는 부분을 의미 footer - 푸터(바닥글,아래글)를 의미
